Lions Gym Website

A modern, responsive fitness website for Lions Gym, built with HTML5,
CSS3, JavaScript, and Bootstrap 5.

Features

🏗️ Structure

1. 3 Complete Pages: Home, About, Membership
2. Responsive Design: Works seamlessly on desktop, tablet, and mobile
3. Semantic HTML5 with accessibility-friendly elements
4. External Stylesheets for clean, modular design
5.Dedicated JavaScript file with animations and interactivity

🎨 Design & Styling

1. Bootstrap 5 Components: Navbar, Carousel, Modals, Offcanvas menu
2. Custom CSS: Modern dark theme with golden accents (#FFD700)
3. Background Images + Overlay Effects for hero and membership sections
4. Animations: Smooth fade-ins, transitions, and hover effects
5. Professional Layout: Flexbox & Grid system for adaptive layout

🚀 Interactive Features

1. Scroll-to-Top Button
2. Smooth Scrolling Navigation
3. Animation on Scroll for feature cards, mission cards, and pricing
    cards
4. Membership Modal Form with validation & AJAX submission
5. Contact Form with client-side validation and toast notifications
6. Hero Section Animation with sequential fade-ups
7. Navbar Scroll Effect with background blur

📱 Responsive Components

1. Mobile Navigation: Offcanvas menu for smaller devices
2. Flexible Layouts: Flexbox and grid for responsive sections
3. Auto-resizing Forms: Dynamic textarea adjustments
4. Adaptive Images with object-fit for trainers carousel

🎬 Multimedia & Enhancements

1. Trainer Carousel with images and bios
2. Embedded Google Map in Contact section
3. Social Media Integration: Facebook, Instagram, Twitter, YouTube
4. Dynamic Counters (stats support included)
5. Form Submission support for backend integration

Technical Implementation

HTML5

1. Semantic structure (<header>, <nav>, <section>, <footer>)
2. Accessible labels and ARIA support
3. Meta viewport for responsive scaling

CSS3

1. Custom dark theme with golden highlights
2. Flexbox & Grid-based responsive layout
3. Keyframe animations for hero and section reveals
4. Styled modals and ribbons for pricing plans

JavaScript (java.js)

1. Scroll-to-top button
2. Navbar scroll behavior with blur
3. Animation on Scroll using Intersection Observer
4. Contact & Membership Form validation + AJAX submit
5. Toast Notifications for user feedback
6. Utility Functions (debounce, throttle, parallax, auto-resize)

Bootstrap 5

1. Offcanvas mobile menu
2. Navbar, Carousel, Modals
3. Responsive grid system

Browser Support

1. Chrome
2. Firefox 
3. Safari 
4. Edge 
5. Mobile browsers (iOS Safari, Chrome Mobile)

Accessibility
1.Semantic HTML structure
2. Alt text for images
3. ARIA attributes for forms & modals
4. Keyboard navigation support
5. Contrast-friendly color scheme
